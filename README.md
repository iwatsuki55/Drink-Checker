# 🥃 NIGHTCAP

**怒らへん晩酌＆体重ログ** — 説教はせえへん。ただ、正直につけるだけ。

晩酌と体重を淡々と記録して、体重は **7日移動平均** の「流れ」で見るプライベート・トラッカー。
「飲み過ぎ」みたいな警告は一切出さへん。判断は本人に任せる、それだけ。

👉 **[使ってみる](https://iwatsuki55.github.io/nightcap/)**

## なんでこれを作ったか

食事管理アプリはお酒に厳しい。怒られるから記録せえへん → データが歪む → 管理にならへん。
NIGHTCAP は逆。ジャッジせず淡々と記録するだけやから、こっそりやのうて正直に入力できる。
事実が見えると、人は勝手に調整しはじめる。罪悪感より、よっぽど続く。

## できること

- 体重とお酒を一行で記録（醸造酒 / 蒸留酒で分類）
- 各酒に標準1杯あたりの **kcal目安**（日本食品標準成分表ベース）
- 体重は **7日移動平均** の太線で表示。日々の上下に一喜一憂させへん
- 晩酌は控えめな棒グラフで。**杯数 / kcal** を切り替え可
- 統計：記録日数・体重トレンド・休肝日・晩酌カロリー（事実だけ）
- **CSV書き出し**（種類別＋est_kcal列）。筋トレアプリのデータと並べて分析もできる

## プライバシー

データは **あなたの端末のブラウザ（localStorage）のなかだけ** に保存されます。
サーバーには一切送信されません。誰にも見られません。だから安心して正直に。
※ ブラウザのデータを消すと記録も消えます。大事なら CSV で書き出しを。

## 公開のしかた（GitHub Pages）

1. リポジトリを作成し、`index.html`・`ogp.png` を置く
2. **Settings → Pages → Source** を `main` ブランチのルートに設定
3. 数分で `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開
4. URL を友人に配るだけ。各自が自分の端末でプライベートに使えます

## ことわり

- カロリーは標準的な1杯あたりの **目安**。実際の量・銘柄で前後します
- 健康管理の参考用。医療・栄養指導を置き換えるものではありません
